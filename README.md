# منصة الفلاح الجزائري

منصة شاملة ومتكاملة للفلاح الجزائري تجمع جميع احتياجات القطاع الفلاحي في مكان واحد.

## المميزات

### 1. سوق المعدات الزراعية
- بيع وشراء المعدات الزراعية
- إيجار المعدات (جرارات، محاريث، حصادات، إلخ)
- بيوت بلاستيكية وأنظمة الري

### 2. إدارة العمال
- البحث عن عمال زراعيين مؤهلين
- عرض ملفات العمال مع التقييمات
- التواصل المباشر مع العمال

### 3. التدريب والتكوين
- دورات تدريبية في مختلف المجالات الزراعية
- محتوى تعليمي باللغة العربية
- شهادات معتمدة

### 4. سوق الأراضي والمنشآت
- بيع وشراء الأراضي الزراعية
- إيجار المنشآت الزراعية
- معلومات تفصيلية عن كل عقار

### 5. الأدوية والأسمدة
- بيع الأسمدة والمبيدات
- معلومات عن الاستخدام الصحيح
- أسعار تنافسية

### 6. نظام المواقع
- اختيار من جميع ولايات الجزائر (58 ولاية)
- تحديد البلدية بدقة
- بحث حسب الموقع

## التقنيات المستخدمة

- **Next.js 15** - إطار عمل React الحديث
- **TypeScript** - لغة برمجة قوية ومكتوبة
- **Tailwind CSS** - تصميم عصري ومتجاوب
- **Shadcn/ui** - مكونات واجهة مستخدم احترافية
- **Capacitor** - تحويل إلى تطبيق موبايل أصلي

## البدء

### التطوير المحلي

\`\`\`bash
# تثبيت المكتبات
npm install

# تشغيل السيرفر المحلي
npm run dev

# فتح المتصفح على
http://localhost:3000
\`\`\`

### البناء للإنتاج

\`\`\`bash
# بناء التطبيق
npm run build

# تشغيل النسخة المبنية
npm start
\`\`\`

### التحويل لتطبيق موبايل

راجع ملف [CAPACITOR_SETUP.md](./CAPACITOR_SETUP.md) للتعليمات الكاملة.
